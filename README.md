# vue-form-validation

This repo contains an example of form validation in Vue.js using [Vuelidate](https://vuelidate-next.netlify.app/).

## Installation

Clone the repo

```bash
git clone https://github.com/bangunbagustapa/vue-form-validation.git
```

Change directory

```bash
cd vue-form-validation
```

Checkout to the desired branch listed [here](https://github.com/bangunbagustapa/vue-form-validation/branches)

```bash
git checkout <branch>
```

Install the dependencies

```bash
npm install
```

## Running locally

Compiles and hot-reloads for development

```bash
npm run serve
```
